## Python Basics Project 1 - Word Counter

A simple Python program that allows the user to perform multiple text analysis tasks on a sentence.

## Features
- Count the number of words
- Count the number of letters (excluding spaces and punctuation)
- Count how many words start with a specific letter
- Count how many words are longer than 5 characters

## Concepts Used
- Functions
- Loops
- Strings
- Lists
- Dictionaries
- User Input

## How to Run
1. Make sure Python is installed
2. Run the program:

## Notes
- Words are counted based on space separation.
- Punctuation attached to words is considered part of the word.
